# Hanzi

Hanzi is Chinese character manipulating module. At present it decomposes single characters into two radicals.

Currently the data was generated by etc etc.

How to use
========================

```javascript
	//Require
	var hanzi = require("hanzi");
	//Initiate
	hanzi.start();
	//Decompose Characters - Returns an Array
	hanzi.decompose('æ±‰');
```
